# kazi

[![PyPI](https://img.shields.io/pypi/v/kazi)](https://pypi.org/project/kazi/)
[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)

Batteries-included meta-package for kazi. Installs `kazi-core` and all its layers in one command.

```bash
pip install kazi[anthropic]
pip install kazi[openai]
pip install kazi[all]
```

This package contains no Python code — it is a thin install wrapper. All functionality lives in `kazi-core`.

For documentation, examples, and configuration reference see the [kazi repository](https://github.com/erose2502/kazi-core).
